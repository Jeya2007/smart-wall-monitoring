WHAT IS THIS SYSTEM?
Smart Wall
Monitoring System
DEFINITION
An autonomous IoT device that attaches to walls and structures — continuously sensing, analysing, and alerting about structural health conditions without human intervention. It detects cracks, vibration, moisture & temperature in real time.
📡 Sensing Layer
4 sensors capture crack width, vibration amplitude, moisture %, and temperature — simultaneously, every 5 seconds.
🧠 Processing Brain
Arduino Uno + NodeMCU read all sensor inputs, apply threshold logic, and decide when danger exists.
🚨 Alert Engine
Buzzer activates, SMS sent via GSM, cloud dashboard flags — all within 5 seconds of threshold breach.
☁️ Cloud Analytics
ThingSpeak stores 30-day history. MATLAB visualisation. Accessible globally from any device.
